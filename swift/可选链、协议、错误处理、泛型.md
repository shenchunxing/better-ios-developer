8.表达式模式（Expression Pattern）
---------------------------

表达式模式用在`case`中

swift

复制代码

`let point = (1, 2)
switch point {
case (0, 0):
print("(0, 0) is at the origin.")
case (-2...2, -2...2):
print("((point.0), (point.1) is near the origin.")
default:
print("The point is at ((point.0), (point.1).")
} // (1, 2) is near the origin.` 

通过反汇编，我们可以看到其内部会调用`~=运算符`来计算`(-2...2, -2...2)`这个区间
